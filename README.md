# Hangul

This gem transcript Hangul into English.

## Installation

Add this line to your application's Gemfile:

    gem 'hangul'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hangul

## Usage

Use method "String.to_roman":

    "'안녕 하세요'.to_roman"
    => "annyeong haseyo"
